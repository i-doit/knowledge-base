# Formular ausfüllen

Mit dem Forms Add-on können Formulare erstellt werden, die von Benutzern -- auch ohne direkten Zugang zur i-doit-Oberfläche -- ausgefüllt werden können. So lassen sich beispielsweise Anfragen für neue Hardware, Onboarding-Prozesse oder Änderungsanträge standardisiert erfassen.

## Vorgehensweise

1. Öffne das Formular über den bereitgestellten Link. Der Link wird vom Ersteller des Formulars geteilt.
2. Fülle die angezeigten Felder aus. Pflichtfelder sind entsprechend gekennzeichnet.
3. Klicke auf **Absenden**, um das Formular abzuschicken.

[![Formular ausfüllen](../../assets/images/de/i-doit-add-ons/forms/ausfuellen/fillout.gif)](../../assets/images/de/i-doit-add-ons/forms/ausfuellen/fillout.gif)

Nach dem Absenden wird automatisch ein neues Objekt in i-doit mit den ausgefüllten Daten angelegt. Welcher Objekttyp erstellt wird und welche Kategorien befüllt werden, hängt von der Konfiguration des Formulars ab.
