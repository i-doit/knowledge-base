# Migration

Dieser Abschnitt beschreibt die Migration von VIVA1 zu VIVA2.

## Voraussetzungen

- VIVA1 und VIVA2 sind installiert. Das IT-Grundschutz-Kompendium ist in VIVA2 importiert

## Schritte

1. Informationsverbünde auswählen
2. Zielobjekt-Gruppen übernehmen
3. Bausteine zuordnen und Vorschläge prüfen
4. Anforderungen zuordnen und Vorschläge prüfen
5. Migration starten und Ergebnis kontrollieren

## Nachbereitung

- Nicht zuordenbare Einträge manuell prüfen
- Stichproben in Reports und im IT-Grundschutz-Check durchführen
