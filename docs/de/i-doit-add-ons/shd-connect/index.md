Dieses Modul benötigt den [**SHD VIEW Client**](https://smdocu.atlassian.net/wiki/spaces/SKB/pages/742260740/SHD+View+Desktop+Client) als Basismodul.

## Installation

Die aktuelle Version des **SHD Connect** Add-On kann unter folgendem Link heruntergeladen werden: [Download](https://smdocu.atlassian.net/wiki/spaces/SKB/pages/6062081/Installation)

Eine Auflistung der verschiedenen Versionen inklusive deren Changelog kann unter folgendem Link eingesehen werden: [SHD Connect](https://smdocu.atlassian.net/wiki/x/KwCcgw)

Nachdem Sie das Add-On heruntergeladen haben können Sie dies bequem über das [Admin-Center](../../administration/admin-center.md) von i-doit hochladen und installieren.
