# Release Notes 38
<!-- cSpell:disable -->
Wir freuen uns, i-doit 38 vorstellen zu können, mit dem wir unseren Fokus auf **Qualität und Kompatibilität** fortsetzen.

Mit dieser Version haben wir offiziell die Kompatibilität mit MariaDB 11.8 implementiert, um den nächsten Schritt zur Anpassung an moderne Systeme mit MariaDB zu gehen. Darüber hinaus enthält diese Version 50 Korrekturen bekannter Probleme.

Eine detaillierte Liste aller Änderungen und Verbesserungen finden Sie im [Changelog](../changelogs/index.md).

## Highlights dieser Version

- 50 **Qualitätsverbesserungen**
- Neue Funktion zur Suche nach MAC-Adressen im Cisco-Format
- **Kompatibilität** mit **MariaDB 11.8**
