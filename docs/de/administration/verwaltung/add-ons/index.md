# Add-ons

Hier können Add-ons konfiguriert werden.

## Die Add-ons sind auf die folgenden Unterabschnitte aufgeteilt

-   [JSON-RPC API](./json-rpc-api/index.md) — i-doit ermöglicht den externen Zugriff auf die IT-Dokumentation über eine application programming interface (API).
-   [Events](./events.md) — Das Event add-on ermöglicht einen hohen Grad an Automatisierung bei der Arbeit mit der IT-Dokumentation.
-   [Gerätetausch](./geraetetausch/index.md) — Ein neuer Switch wird in Betrieb genommen, aber du willst alle Verbindungen beibehalten?
