# Gerätetausch

Ein neuer Switch wird in Betrieb genommen, aber du willst alle Verbindungen beibehalten? Verwende das [==Gerätetausch== Add-on](../../../../i-doit-add-ons/replacement.md), damit du nicht alle Verbindungen neu dokumentieren musst. Dieses Add-on ermöglicht das Austauschen von [Objekten](../../../../grundlagen/struktur-it-dokumentation.md) mit konfigurierbarer Übernahme bestehender Verbindungen.

## Die Add-ons sind in die folgenden Unterpunkte aufgeteilt

-   [Konfiguration](./konfiguration.md) — Für weitere Informationen -> WEITERLESEN
