# Kategorien und Attribute

Hier werden alle verfügbaren [Attribute für die ausgewählte Kategorie](../../../../grundlagen/struktur-it-dokumentation.md) von i-doit aufgelistet.

Du erreichst diese Übersicht unter **Verwaltung → Add-ons → JSON-RPC API → Kategorien und Attribute**. Diese Seite zeigt dir eine Referenzliste aller Kategorien mit ihren jeweiligen Attributen, wie sie über die i-doit JSON-RPC API angesprochen werden können. Für jede Kategorie siehst du die technischen Feldnamen (Konstanten), die du in API-Aufrufen verwenden musst, um Daten zu lesen oder zu schreiben. Diese Übersicht ist besonders hilfreich, wenn du eigene Integrationen oder Skripte entwickelst, die über die API mit i-doit kommunizieren, und du die korrekten Bezeichner für Kategorien und Attribute benötigst.

[![Kategorien und Attribute](../../../../assets/images/de/administration/verwaltung/add-ons/json-rpc-api/kategorien-und-attribute.png)](../../../../assets/images/de/administration/verwaltung/add-ons/json-rpc-api/kategorien-und-attribute.png)

Eine ausführliche Dokumentation zur API findest du unter [JSON-RPC API](../../../../i-doit-add-ons/api/index.md).
