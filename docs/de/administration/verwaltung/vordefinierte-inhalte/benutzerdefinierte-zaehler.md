# Benutzerdefinierte Zähler

Hier kannst du benutzerdefinierte Zähler zu deinem i-doit hinzufügen.

Du erreichst diese Einstellung unter **Verwaltung → Vordefinierte Inhalte → Benutzerdefinierte Zähler**. Mit benutzerdefinierten Zählern legst du fest, wie Objekte automatisch benannt und nummeriert werden, wenn sie einem bestimmten Objekttyp zugeordnet sind. Du kannst Präfixe, Suffixe, die Startnummer und die Schrittweite definieren. So erhält beispielsweise jeder neue Server automatisch eine fortlaufende Bezeichnung wie "SRV-001", "SRV-002" usw. Nutze diese Funktion, wenn du eine einheitliche und automatische Namensvergabe für bestimmte Objekttypen einrichten möchtest.

Eine ausführliche Anleitung findest du unter [Benutzerdefinierte Zähler](../../../grundlagen/benutzerdefinierte-zaehler.md).
