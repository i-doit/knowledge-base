# Verwaltung

Die Verwaltung ist der zentrale Bereich für alle administrativen Tätigkeiten in i-doit. Hier konfigurierst du die [IT-Dokumentation](../../glossar.md) über die Web GUI -- von Benutzereinstellungen bis hin zu Schnittstellen.

## Wo finde ich die Verwaltung?

Du erreichst die Verwaltung über das Zahnrad-Icon in deinem Benutzerbereich. Dieser erscheint, sobald du mit der Maus über deinen Anmeldenamen im oberen rechten Bereich fährst.
