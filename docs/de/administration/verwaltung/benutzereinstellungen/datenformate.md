# Datenformate

Unter **Verwaltung → Benutzereinstellungen → Datenformate** kannst du festlegen, wie Datumswerte, Zahlen und Währungen in der Oberfläche angezeigt werden. Diese Einstellungen gelten nur für deinen eigenen Benutzer und haben keinen Einfluss auf andere Benutzer oder auf die gespeicherten Daten in der Datenbank.

Das **Datumsformat** bestimmt die Reihenfolge von Tag, Monat und Jahr. Das **Numerische Format** steuert, ob ein Punkt oder ein Komma als Dezimaltrennzeichen verwendet wird. Über das **Währungsformat** wählst du die gewünschte Währung und das zugehörige Symbol aus.

![datenformate](../../../assets/images/de/administration/verwaltung/benutzereinstellungen/datenformate-20260407-103211.png)

| Option | Wert |
| ------ | ---- |
| Datumsformat | Englisch/Deutsch |
| Numerisches Format | Englisch/Deutsch |
| Währungsformat | `EUR;€` oder `USD;$` oder `GBP;£` oder `CHF;Sfr.` |
