# Bearbeitungssperre

Unter **Verwaltung → Benutzereinstellungen → Bearbeitungssperre** konfigurierst du den Schutz vor gleichzeitiger Bearbeitung desselben Objekts durch mehrere Benutzer. Die Bearbeitungssperre verhindert, dass mehrere Benutzer gleichzeitig dasselbe Objekt bearbeiten. Sobald ein Benutzer ein Objekt editiert, wird es für andere Benutzer gesperrt. Die Sperre bleibt bestehen, bis der Editiervorgang abgeschlossen oder der konfigurierte Timeout erreicht ist.

Über die Option **Sperren von Objekten im Bearbeitungsmodus aktivieren** schaltest du die Funktion ein oder aus. Der **Timeout** legt fest, nach wie vielen Sekunden eine Sperre automatisch aufgehoben wird, falls ein Benutzer den Bearbeitungsmodus nicht ordnungsgemäß verlässt. Unter **Aktuell gesperrte Objekte** siehst du eine Übersicht aller derzeit gesperrten Objekte und kannst Sperren bei Bedarf manuell aufheben.

[![Bearbeitungssperre](../../../assets/images/de/administration/benutzereinstellungen/bearbeitungssperre.png)](../../../assets/images/de/administration/benutzereinstellungen/bearbeitungssperre.png)

| Option                                               | Wert                        |
| ---------------------------------------------------- | --------------------------- |
| Sperren von Objekten im Bearbeitungsmodus aktivieren | Ja oder Nein                |
| Timeout                                              | 120 Wert in Sekunden        |
| Aktuell gesperrte Objekte                            | Liste der Objekte oder none |
