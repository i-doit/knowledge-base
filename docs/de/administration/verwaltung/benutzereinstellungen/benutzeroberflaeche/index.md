# Benutzeroberfläche

Hier findest du die Einstellungen für die Benutzeroberfläche

[![Benutzeroberfläche](../../../../assets/images/de/administration/benutzereinstellungen/benutzeroberflaeche.png)](/img/assets/images/de/administration/benutzereinstellungen/benutzeroberflaeche.png)

## Abstände

![index](../../../../assets/images/de/administration/verwaltung/benutzereinstellungen/benutzeroberflaeche/index-20260407-103659.png)

| Option | Wert |
| ------ | ---- |
| Zeilenabstände | [Klein oder Mittel oder Groß](../../../../grundlagen/web-gui.md#zeilen-abstande) |
| Abstände in Menübäumen | [Klein oder Mittel oder Groß](../../../../grundlagen/web-gui.md#abstande-in-menubaumen) |
| Abstandhalter darstellen | [Ja oder Nein](../../../../grundlagen/web-gui.md#abstandhalter-darstellen) |

## Hauptmenü

![index](../../../../assets/images/de/administration/verwaltung/benutzereinstellungen/benutzeroberflaeche/index-20260407-103712.png)

| Option | Wert |
| ------ | ---- |
| "my-doit" im Hauptmenü darstellen | Ja oder Nein |
| Objekttyp Gruppe als Dropdown darstellen | Ja oder Nein |

## Kategorien

![index](../../../../assets/images/de/administration/verwaltung/benutzereinstellungen/benutzeroberflaeche/index-20260407-103725.png)

| Option | Wert |
| ------ | ---- |
| Kabelansicht direkt öffnen | Ja oder Nein |

## Objekt-Listen

![index](../../../../assets/images/de/administration/verwaltung/benutzereinstellungen/benutzeroberflaeche/index-20260407-103736.png)

| Option | Wert |
| ------ | ---- |
| Zuletzt verwendeten Listen-Filter merken | 300 In Sekunden (0 = Filter nicht merken) |
| Einträge pro Seite in Objektlisten | 50 |

## Objektnavigation

![index](../../../../assets/images/de/administration/verwaltung/benutzereinstellungen/benutzeroberflaeche/index-20260407-103748.png)

| Option | Wert |
| ------ | ---- |
| Standard Baumansicht | [Objektsicht](../../../../grundlagen/web-gui.md#objektansicht)<br>[Standortsicht](../../../../grundlagen/web-gui.md#standortsicht) |
| Baumansicht Typ | [Physischer Standort oder Logischer Standort oder Kombinierte Ansicht](../../../../grundlagen/web-gui.md#standortsicht) |
| Leere Objekttypen ausblenden | Ja oder Nein |
| Leere Kategorien ausblenden | Ja oder Nein |
