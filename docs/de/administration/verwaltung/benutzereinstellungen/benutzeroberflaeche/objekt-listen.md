# Objekt-Listen

Unter **Verwaltung → Benutzereinstellungen → Benutzeroberfläche → Objekt-Listen** kannst du die Darstellung der [Objekt-Listen](../../../../grundlagen/objekt-liste/index.md) an deine persönlichen Bedürfnisse anpassen. Die Einstellungen gelten nur für deinen eigenen Benutzer.

Du kannst hier beispielsweise die Standard-Sortierung, die Anzahl der angezeigten Einträge pro Seite und die Darstellung von Attributen in der Listenansicht konfigurieren. So behältst du auch bei großen Datenbeständen den Überblick und kannst die Objekt-Listen auf deine tägliche Arbeitsweise abstimmen.
