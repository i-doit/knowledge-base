# Benutzersprache

Unter **Verwaltung → Benutzereinstellungen → Benutzersprache** kannst du die Sprache der i-doit-Oberfläche für deinen Benutzer festlegen. Die Einstellung betrifft Menüs, Kategoriebeschriftungen, Systemmeldungen und alle weiteren Oberflächentexte.

Wenn du die Option **Sprache durch Browser wählen** aktivierst, übernimmt i-doit automatisch die im Browser eingestellte Sprache. Ist diese Option deaktiviert, wird stattdessen die manuell gewählte Sprache verwendet. Neben Deutsch und Englisch stehen auch benutzerdefinierte Sprachen zur Verfügung, die über die [Sprachprofile](../datenansicht/sprachprofile.md) angelegt werden können.

![benutzersprache](../../../assets/images/de/administration/verwaltung/benutzereinstellungen/benutzersprache-20260407-103241.png)

| Option | Wert |
| ------ | ---- |
| Sprache durch Browser wählen | Ja oder Nein |
| Sprache | Englisch<br>Deutsch<br>Benutzerdefinierte Sprache |
