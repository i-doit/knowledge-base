# Berechtigungen

Über die Berechtigungen steuerst du, welche Benutzer auf welche Bereiche von i-doit zugreifen dürfen. Ein Praxisbeispiel findest du auf unserem [Blog](https://www.i-doit.com/blog/das-rechtesystem-in-i-doit/).

## Rechtevergabe

Hier konfigurierst du die Berechtigungen für einzelne i-doit-Benutzer und -Gruppen. → [WEITERLESEN](../../effizientes-dokumentieren/rechteverwaltung/index.md)

## Rechtesystemfehler lösen

Mit dieser Funktion setzt du die Rechte des Admin-Users auf den Auslieferungszustand zurück. Nutze sie, wenn du dich durch eine Fehlkonfiguration ausgesperrt hast.
