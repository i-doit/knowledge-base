# Objekt-Browser Konfiguration

Der Objekt-Browser in i-doit ermöglicht es, Objekte tabellarisch darzustellen. Die Konfiguration des Objekt-Browsers erlaubt es Administratoren, die Anzeige und Funktionalität an die Bedürfnisse der Benutzer anzupassen.

Es lassen sich verschiedene Einstellungen vornehmen, wie z.B. die Auswahl der anzuzeigenden Spalten, die Sortierreihenfolge und Filteroptionen.

[![Objekt-Browser](../../../assets/images/de/administration/verwaltung/datenansicht/objekt-browser-konfigurieren/1-obk.png)](../../../assets/images/de/administration/verwaltung/datenansicht/objekt-browser-konfigurieren/1-obk.png)

Dazu muss zunächst ein Attribut aus einer Kategorie ausgewählt werden, das für den Objekte-Browser konfiguriert werden soll.

[![Objekt-Browser](../../../assets/images/de/administration/verwaltung/datenansicht/objekt-browser-konfigurieren/2-obk.png)](../../../assets/images/de/administration/verwaltung/datenansicht/objekt-browser-konfigurieren/2-obk.png)

Diese Attribute werden dann im Objekt-Browser als Spalten angezeigt. Es ist möglich, mehrere Attribute auszuwählen, um eine umfassende Ansicht der Objektdaten zu erhalten.

[![Objekt-Browser](../../../assets/images/de/administration/verwaltung/datenansicht/objekt-browser-konfigurieren/3-obk.png)](../../../assets/images/de/administration/verwaltung/datenansicht/objekt-browser-konfigurieren/3-obk.png)
