# Benutzerdefinierte Kategorien

Hier kannst du die benutzerdefinierten Kategorien organisieren und bearbeiten.

Du erreichst diese Einstellung unter **Verwaltung → Datenstruktur → Benutzerdefinierte Kategorien**. Auf dieser Seite siehst du eine Übersicht aller selbst erstellten Kategorien und kannst neue anlegen, bestehende bearbeiten oder löschen. Benutzerdefinierte Kategorien ermöglichen es dir, die Datenstruktur von i-doit an deine individuellen Anforderungen anzupassen, indem du eigene Felder und Feldtypen definierst. So kannst du Informationen dokumentieren, die von den Standardkategorien nicht abgedeckt werden. Nutze diese Funktion, wenn du zusätzliche Datenfelder für bestimmte Objekttypen benötigst.

Eine ausführliche Anleitung zum Erstellen und Konfigurieren findest du unter [Benutzerdefinierte Kategorien](../../../grundlagen/benutzerdefinierte-kategorien.md).
