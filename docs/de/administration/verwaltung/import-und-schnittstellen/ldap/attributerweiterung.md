# LDAP Attributerweiterung

Mit dieser Funktion können Sie die Namen der Felder aus der LDAP-Konfigurationsdatei anpassen. Außerdem wird die Einstellung benötigt, wenn Sie die Eindeutige Kennung für Personen über ein anderes Attribut festlegen möchten.

[![ldap-attribut-erweiterung](../../../../assets/images/de/administration/verwaltung/import-und-schnittstellen/ldap/1-ldap.png)](../../../../assets/images/de/administration/verwaltung/import-und-schnittstellen/ldap/1-ldap.png)

Über **Verwaltung → Import und Schnittstellen → LDAP → Attributerweiterung** können Sie weitere Felder für die Kategorie Person konfigurieren. Sobald ein Feld-Name definiert wurde, werden die Felder in der Kategorie **Personen -> Personen** angezeigt. Die Befüllung findet dann über den zugehörigen Key über eine `ldap.ini` statt.
