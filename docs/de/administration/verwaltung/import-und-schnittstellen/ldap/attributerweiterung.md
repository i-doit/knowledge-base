# LDAP Attributerweiterung

Mit dieser Funktion kannst du die Namen der Felder aus der LDAP-Konfigurationsdatei anpassen.

Du erreichst diese Einstellung unter **Verwaltung → Import und Schnittstellen → LDAP → Attributerweiterung**. Hier kannst du zusätzliche LDAP-Attribute definieren, die über das Standard-Mapping hinausgehen. Wenn dein Verzeichnisdienst benutzerdefinierte Attribute enthält (z. B. eigene Schema-Erweiterungen im Active Directory), kannst du diese hier hinzufügen und ihnen einen lesbaren Feldnamen in i-doit zuweisen. Damit stehen dir beim LDAP-Import weitere Felder zur Verfügung, die automatisch in die Personenstammdaten übernommen werden. Nutze diese Funktion, wenn du spezielle Attribute aus deinem LDAP-Verzeichnis in i-doit abbilden möchtest.

Eine ausführliche Dokumentation zur LDAP-Anbindung findest du unter [LDAP-Verzeichnis](../../../../benutzerauthentifizierung-und-verwaltung/ldap-verzeichnis/index.md).
