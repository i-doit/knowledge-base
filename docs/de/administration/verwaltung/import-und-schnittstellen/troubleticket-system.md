# TroubleTicket-System (TTS)

Hier kannst du das TroubleTicket-System (TTS) Interface konfigurieren.

Du erreichst diese Einstellung unter **Verwaltung → Import und Schnittstellen → TroubleTicket-System (TTS)**. Über diese Schnittstelle verbindest du i-doit mit einem externen Ticketsystem wie z. B. Request Tracker (RT), OTRS oder iTop. Dabei legst du die URL zum TTS fest und konfigurierst die Verknüpfungsparameter, damit aus i-doit heraus direkt auf Tickets verwiesen werden kann. Wenn du in der Kategorie "TroubleTicket" eines Objekts eine Ticket-ID hinterlegst, wird daraus automatisch ein klickbarer Link zum jeweiligen Ticket im externen System generiert. Nutze diese Funktion, wenn du dein Incident- oder Service-Management mit deiner IT-Dokumentation verknüpfen möchtest.
