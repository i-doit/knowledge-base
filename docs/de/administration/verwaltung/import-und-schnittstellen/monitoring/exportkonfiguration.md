# Exportkonfiguration

Definiere die [Exportparameter](../../../../i-doit-add-ons/nagios.md) für dein Monitoring-System.

Du erreichst diese Einstellung unter **Verwaltung → Import und Schnittstellen → Monitoring → Exportkonfiguration**. Hier legst du fest, wie i-doit die Monitoring-Konfiguration (z. B. für Nagios oder Checkmk) exportiert. Du kannst Export-Profile anlegen und konfigurieren, in denen du den Exportpfad, den Typ und weitere Parameter bestimmst. Damit generiert i-doit aus den in der CMDB hinterlegten Monitoring-Daten automatisch die passenden Konfigurationsdateien für dein Monitoring-System. Nutze diese Funktion, wenn du deine Monitoring-Konfiguration direkt aus der IT-Dokumentation heraus pflegen und automatisiert bereitstellen möchtest.
