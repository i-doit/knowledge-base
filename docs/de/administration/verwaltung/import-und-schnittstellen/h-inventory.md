# H-Inventory

Hier kannst du die Objekt-Matching-Profile für den h-Inventory Import konfigurieren und aktivieren.

Du erreichst diese Einstellung unter **Verwaltung → Import und Schnittstellen → H-Inventory**. Beim Import von Hardware-Inventardaten über h-Inventory muss i-doit erkennen können, ob ein importiertes Objekt bereits in der CMDB existiert oder ob es neu angelegt werden soll. Dafür dienen die Objekt-Matching-Profile: Du definierst hier, nach welchen Kriterien (z. B. Hostname, Seriennummer oder MAC-Adresse) i-doit vorhandene Objekte identifiziert und zuordnet. So vermeidest du Duplikate beim wiederholten Import. Nutze diese Seite, wenn du den h-Inventory Import einrichtest oder das Matching-Verhalten anpassen möchtest.

Eine ausführliche Dokumentation findest du unter [h-Inventory](../../../daten-konsolidieren/h-inventory.md).
