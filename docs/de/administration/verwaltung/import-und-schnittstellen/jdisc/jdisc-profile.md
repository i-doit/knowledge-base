# JDisc-Profiles

!!! warning "Bitte erstelle vor jeder Änderung an einer Schnittstelle/Import ein vollständiges Backup. Falls das Ergebnis nicht zufriedenstellend ist, kann dieses dann wiederhergestellt werden"

Unter **Verwaltung → Import und Schnittstellen → JDisc → JDisc-Profiles** konfigurierst du die Importprofile, die festlegen, wie Daten aus JDisc Discovery in die i-doit CMDB übernommen werden. Mit den JDisc-Profilen kannst du Objekte und ihre Attribute aus der JDisc-Datenbank mit den gleichen Objekten in i-doit verbinden.

In einem Profil definierst du unter anderem, welche JDisc-Gerätetypen welchen i-doit-Objekttypen zugeordnet werden, welche Kategorien beim Import befüllt werden sollen und wie mit bereits vorhandenen Daten umgegangen wird (erstellen, aktualisieren oder überspringen). Du kannst mehrere Profile für unterschiedliche Import-Szenarien anlegen -- zum Beispiel ein Profil für Server und ein weiteres für Netzwerkgeräte. Die ausführliche Dokumentation findest du unter [JDisc Discovery](../../../../_archiv/jdisc-discovery.md).
