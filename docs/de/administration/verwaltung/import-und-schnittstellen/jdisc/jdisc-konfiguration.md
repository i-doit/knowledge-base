# JDisc Konfiguration

!!! warning "Bitte erstelle vor jeder Änderung an einer Schnittstelle/Import ein vollständiges Backup. Falls das Ergebnis nicht zufriedenstellend ist, kann dieses dann wiederhergestellt werden"

Unter **Verwaltung → Import und Schnittstellen → JDisc → JDisc Konfiguration** stellst du die Verbindung zwischen i-doit und deinem JDisc Discovery Server her. Trage den Hostnamen oder die IP-Adresse des JDisc-Servers sowie die Zugangsdaten für die JDisc-Datenbank ein. Über die Schaltfläche **Verbindung testen** prüfst du, ob die Verbindung erfolgreich aufgebaut werden kann.

Die JDisc-Integration ermöglicht es, automatisch erkannte Geräte, Netzwerkverbindungen und Softwareinstallationen aus der JDisc-Datenbank in die i-doit CMDB zu importieren. Die ausführliche Dokumentation zur JDisc-Anbindung findest du unter [JDisc Discovery](../../../../_archiv/jdisc-discovery.md).
