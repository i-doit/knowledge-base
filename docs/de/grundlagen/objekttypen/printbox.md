# Objekttyp: Printbox

Der Objekttyp **Printbox** dokumentiert Druckserver, die mehrere Drucker im Netzwerk zentral verwalten. Eine Printbox ist häufig ein dediziertes Gerät oder eine Softwarekomponente.

## Verwendung

- **Druckinfrastruktur**: Dokumentiere Printserver mit den angebundenen Druckern.
- **Netzwerk-Integration**: Hinterlege IP-Adresse und Netzwerkinformationen.
