# Objekttyp: Host

Der Objekttyp **Host** dokumentiert entfernte Rechner, die nicht direkt zur IT-Infrastruktur des Unternehmens gehören, aber für die Dokumentation relevant sind – z.B. externe DNS-Server oder Cloud-Dienste.

## Verwendung

- **Externe Systeme**: Dokumentiere externe Server oder Cloud-Ressourcen als Host-Objekte.
- **DNS-Server**: Erfasse externe DNS-Resolver mit IP-Adresse und Betreiber.
- **Abhängigkeiten**: Verknüpfe interne Systeme mit externen Hosts, die sie nutzen.
