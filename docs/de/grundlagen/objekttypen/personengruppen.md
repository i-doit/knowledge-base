# Objekttyp: Personengruppen

Der Objekttyp **Personengruppen** dokumentiert Gruppen von Personen, z.B. Abteilungen, Teams oder Berechtigungsgruppen. Personengruppen vereinfachen die Zuweisung von Verantwortlichkeiten an mehrere Personen gleichzeitig.

## Verwendung

- **Abteilungsstruktur**: Bilde die Organisationsstruktur durch Personengruppen ab.
- **Gruppenverantwortung**: Weise Personengruppen als verantwortliche Stelle einem Objekt zu.
- **Berechtigungsgruppen**: Nutze Personengruppen für rollenbasierte Zugriffskontrollen.
