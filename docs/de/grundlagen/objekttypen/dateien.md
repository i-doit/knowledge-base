# Objekttyp: Dateien

Der Objekttyp **Dateien** ermöglicht das Hochladen und Verwalten von Dokumenten direkt in i-doit. Dateien können mit beliebigen anderen Objekten verknüpft werden und dienen als zentrales Dokumenten-Repository.

## Verwendung

- **Dokumenten-Repository**: Lade Handbücher, Datenblätter, Konfigurationsbackups oder Zertifikate hoch.
- **Versionierung**: Pflege verschiedene Versionen eines Dokuments mit Versionshistorie.
- **Verknüpfung mit Objekten**: Weise Dateien über die Dateizuweisung-Kategorie beliebigen Objekten zu.
