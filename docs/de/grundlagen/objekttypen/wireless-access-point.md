# Objekttyp: Wireless Access Point

Der Objekttyp **Wireless Access Point** dokumentiert WLAN-Access-Points. Access Points verbinden kabellose Endgeräte mit dem kabelgebundenen Netzwerk und bilden die Grundlage der WLAN-Infrastruktur-Dokumentation.

## Verwendung

- **WLAN-Inventar**: Erfasse Hersteller, Modell und Firmware-Version aller Access Points.
- **Standort**: Weise den Access Point einem Raum oder Gebäude zu für eine flächendeckende WLAN-Planung.
- **Netzwerkanbindung**: Dokumentiere den Switch-Port, an dem der Access Point angeschlossen ist.
- **Controller-Zuweisung**: Verknüpfe den Access Point mit dem zugehörigen Access-Point-Controller.
