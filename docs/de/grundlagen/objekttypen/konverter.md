# Objekttyp: Konverter

Der Objekttyp **Konverter** dokumentiert Komponenten zur Signal- oder Medienkonvertierung, z.B. Medienkonverter von Kupfer auf Glasfaser. Konverter kommen überall dort zum Einsatz, wo unterschiedliche Übertragungsmedien verbunden werden müssen.

## Verwendung

- **Netzwerk-Infrastruktur**: Erfasse Konverter als Bestandteil der Verkabelungsinfrastruktur.
- **Kabelverbindungen**: Dokumentiere die angeschlossenen Kabel auf beiden Seiten.
- **Standort**: Weise den Konverter einem Rack oder Raum zu.
