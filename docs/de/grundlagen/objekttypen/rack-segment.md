# Objekttyp: Rack-Segment

Der Objekttyp **Rack-Segment** ermöglicht die Unterteilung eines Racks in logische Segmente, z.B. für verschiedene Mandanten oder Abteilungen in einem geteilten Schrank.

## Verwendung

- **Mandantenfähige Racks**: Unterteile einen Rack in Segmente für verschiedene Abteilungen oder Kunden.
- **Organisations-Zuweisung**: Weise Rack-Segmente einer Organisation oder Kostenstelle zu.
