# Objekttyp: Blade Chassis

Der Objekttyp **Blade Chassis** dokumentiert das Gehäuse, das mehrere Blade Server aufnimmt. Das Chassis stellt die gemeinsame Infrastruktur – Stromversorgung, Kühlung, Netzwerk-Backplane – für alle eingesetzten Blade Module bereit.

## Verwendung

- **Chassis-Übersicht**: Dokumentiere alle Steckplätze und welche Blade Server darin verbaut sind. i-doit zeigt eine grafische Belegungsansicht.
- **Rack-Integration**: Weise das Blade Chassis einem Rack zu und hinterlege die belegten Höheneinheiten.
- **Netzwerk-Anbindung**: Dokumentiere die Uplink-Ports des Chassis zu Switches.
- **Kapazitätsplanung**: Erkenne freie Blade-Steckplätze und plane Erweiterungen vorausschauend.
