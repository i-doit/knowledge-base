# Objekttyp: Objektgruppe

Der Objekttyp **Objektgruppe** ermöglicht die logische Zusammenfassung beliebiger Objekte. Eine Objektgruppe enthält keine eigenen Felder – sie dient ausschließlich als Container für andere Objekte.

## Verwendung

- **Logische Gruppen**: Fasse zusammengehörende Objekte in einer Gruppe zusammen, z.B. alle Server eines Projekts.
- **Reports**: Nutze Objektgruppen als Selektion für den Report Manager.
- **Überblick**: Erstelle Gruppen für temporäre Arbeitspakete oder Projekte.
