# Objekttyp: Virtueller Host

Der Objekttyp **Virtueller Host** dokumentiert Hypervisoren – Systeme, die virtuelle Maschinen hosten. Ein virtueller Host ist die physikalische oder softwarebasierte Plattform, auf der VMs laufen.

## Verwendung

- **Hypervisor-Inventar**: Dokumentiere VMware ESXi, Hyper-V, KVM und andere Hypervisoren.
- **VM-Zuweisung**: Verknüpfe den virtuellen Host mit den darauf laufenden virtuellen Servern und Clients.
- **Kapazitätsplanung**: Erkenne per Report Manager, wie viele VMs auf jedem Host laufen und wie die Ressourcen verteilt sind.
