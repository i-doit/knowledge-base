# Objekttyp: Virtueller Client

Der Objekttyp **Virtueller Client** dokumentiert virtuelle Desktop-Umgebungen (VDI), die auf einem Virtuellen Host betrieben werden. Virtuelle Clients werden typischerweise in VDI-Infrastrukturen wie VMware Horizon oder Citrix Virtual Desktops eingesetzt.

## Verwendung

- **VDI-Inventar**: Dokumentiere alle virtuellen Desktops mit Host-Zuweisung und Benutzerverknüpfung.
- **Benutzerzuweisung**: Weise jedem virtuellen Client den nutzenden Mitarbeiter zu.
- **Ressourcenplanung**: Behalte über Reports den Überblick über zugewiesene Ressourcen pro VDI-Host.
