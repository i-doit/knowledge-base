# Objekttyp: FC-Switch

Der Objekttyp **FC-Switch** dokumentiert Fibre-Channel-Switches, die in Storage Area Networks (SAN) eingesetzt werden. FC-Switches verbinden Server und Speichersysteme über das Hochgeschwindigkeits-Glasfaserprotokoll.

## Verwendung

- **SAN-Topologie**: Dokumentiere die FC-Switches als zentrale Verbindungskomponente zwischen Servern und Speichersystemen.
- **Port-Zuweisung**: Weise FC-Ports den angeschlossenen HBAs von Servern oder dem Speichersystem zu.
- **SAN-Zoning**: Verknüpfe den FC-Switch mit SAN-Zoning-Objekten für eine vollständige Zone-Dokumentation.
- **Redundanz**: Dokumentiere redundante FC-Switch-Pfade für die Hochverfügbarkeits-Analyse.
