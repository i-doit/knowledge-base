# Objekttyp: Datenbankschema

Der Objekttyp **Datenbankschema** dokumentiert einzelne Datenbanken (Schemata) innerhalb einer Datenbankinstanz. Ein Schema ist die logische Containerstruktur, die Tabellen, Views und Prozeduren enthält.

## Verwendung

- **Schema-Inventar**: Erfasse alle Datenbankschemata mit Instanzzuordnung und verantwortlicher Applikation.
- **Anwendungszuordnung**: Dokumentiere welche Anwendung welches Schema verwendet.
