# Objekttyp: DBMS

Der Objekttyp **DBMS** dokumentiert Datenbank-Management-Systeme wie MariaDB, PostgreSQL, Oracle oder MS SQL Server. Ein DBMS-Objekt repräsentiert die Datenbankengine, die auf einem Server installiert ist.

## Verwendung

- **Datenbankinfrastruktur**: Dokumentiere alle DBMS-Installationen mit Hersteller, Version und Installationsserver.
- **Datenbankinstanzen**: Verknüpfe das DBMS mit den zugehörigen Datenbankinstanzen.
- **Abhängigkeiten**: Erfasse welche Anwendungen von einem DBMS abhängen für Impact-Analysen.
