# Objekttyp: Stadt

Der Objekttyp **Stadt** bildet die zweite Ebene der Standort-Hierarchie. Städte werden einem Land zugeordnet und enthalten Gebäude.

## Verwendung

- **Standort-Hierarchie**: Ordne Städte einem Land zu, um die geografische Struktur abzubilden.
- **Multi-Standort**: Dokumentiere mehrere Büros, Niederlassungen oder Rechenzentren in verschiedenen Städten.
