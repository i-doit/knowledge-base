# Objekttyp: VRRP

Der Objekttyp **VRRP** dokumentiert VRRP-Instanzen (Virtual Router Redundancy Protocol). VRRP ermöglicht automatisches Router-Failover durch eine virtuelle IP-Adresse, die von einem aktiven Router übernommen wird.

## Verwendung

- **Redundante Gateway-Dokumentation**: Dokumentiere virtuelle IP-Adressen und die beteiligten Router.
- **Verfügbarkeits-Analyse**: Nutze VRRP-Objekte für Impact-Analysen bei Router-Ausfällen.
