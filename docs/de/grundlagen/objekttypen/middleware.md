# Objekttyp: Middleware

Der Objekttyp **Middleware** dokumentiert Software, die zwischen Betriebssystem und Anwendungen vermittelt – z.B. Applikationsserver, Message-Broker oder .NET-Framework. Middleware ist häufig Grundvoraussetzung für den Betrieb von Anwendungen.

## Verwendung

- **Middleware-Inventar**: Erfasse alle eingesetzten Middleware-Komponenten mit Version und Installationsort.
- **Abhängigkeiten**: Dokumentiere welche Anwendungen die Middleware benötigen.
- **Lifecycle-Management**: Tracke End-of-Support-Daten für verwendete Middleware-Versionen.
