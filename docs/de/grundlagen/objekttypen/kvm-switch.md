# Objekttyp: KVM-Switch

Der Objekttyp **KVM-Switch** dokumentiert Komponenten zur entfernten Bereitstellung von Eingabegeräten (Keyboard, Video, Mouse). KVM-Switches ermöglichen die Verwaltung mehrerer Server über eine einzige Tastatur-Monitor-Maus-Kombination.

## Verwendung

- **Server-Zuweisung**: Dokumentiere welche Server an welchen Ports des KVM-Switches angeschlossen sind.
- **Rack-Integration**: Weise den KVM-Switch einem Rack zu.
- **Out-of-Band-Management**: Dokumentiere KVM-Switches als Teil der Out-of-Band-Management-Infrastruktur.
