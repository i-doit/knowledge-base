# Objekttyp: Betriebssystem

Der Objekttyp **Betriebssystem** dokumentiert Betriebssysteminstanzen wie Windows Server, Ubuntu Linux oder VMware ESXi. Ein Betriebssystem-Objekt repräsentiert eine konkrete OS-Version und verknüpft sich mit den Systemen, auf denen es installiert ist.

## Verwendung

- **OS-Inventar**: Erfasse alle eingesetzten Betriebssysteme mit Version, Patchlevel und Supportende.
- **Systemzuordnung**: Verknüpfe das Betriebssystem mit den Servern, Clients oder VMs, auf denen es läuft.
- **Patch-Management**: Dokumentiere den Patchstand und erkenne per Report Manager Systeme mit veralteten OS-Versionen.
- **EOL-Tracking**: Hinterlege das End-of-Support-Datum und erhalte frühzeitig Hinweise auf auslaufende Betriebssysteme.
