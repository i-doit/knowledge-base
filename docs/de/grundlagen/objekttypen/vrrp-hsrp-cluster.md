# Objekttyp: VRRP/HSRP Cluster

Der Objekttyp **VRRP/HSRP Cluster** dokumentiert hochverfügbare Router-Gruppen, die die Protokolle VRRP, HSRP oder CARP verwenden. Er bietet eine übergeordnete Sicht auf redundante Gateway-Konfigurationen.

## Verwendung

- **Redundante Gateways**: Fasse mehrere Router, die zusammen ein redundantes Gateway bilden, in einem Cluster-Objekt zusammen.
- **Protokoll-Dokumentation**: Erfasse ob VRRP, HSRP oder CARP eingesetzt wird.
