# Objekttyp: Access Point Controller

Der Objekttyp **Access Point Controller** dokumentiert zentrale Controller, die mehrere Wireless Access Points verwalten und konfigurieren. Der Controller übernimmt WLAN-Richtlinien, Roaming und Lastverteilung für alle angebundenen Access Points.

## Verwendung

- **WLAN-Infrastruktur**: Dokumentiere den Controller als zentralen Verwaltungsknoten der WLAN-Infrastruktur.
- **Access-Point-Zuweisung**: Verknüpfe den Controller mit allen verwalteten Access Points.
- **Netzwerk und Standort**: Weise den Controller seinem Rack-Standort und dem Managementnetz zu.
