# Objekttyp: Switch Chassis

Der Objekttyp **Switch Chassis** dokumentiert modulare Switch-Gehäuse, in die einzelne Linecards eingesetzt werden. Switch-Chassis werden in großen Netzwerken als Core- oder Distribution-Switches eingesetzt.

## Verwendung

- **Chassis-Belegung**: Dokumentiere alle eingesetzten Module und Linecards mit ihren Steckplatznummern.
- **Port-Inventar**: Erfasse alle Ports der eingesetzten Module über die Netzwerkport-Kategorie.
- **Rack-Integration**: Weise das Switch-Chassis einem Rack zu.
