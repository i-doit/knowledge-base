# Objekttyp: Monitor

Der Objekttyp **Monitor** dokumentiert externe Displays, die an Clients oder Server angeschlossen sind. Monitore werden typischerweise einem Arbeitsplatz oder einem Client zugeordnet.

## Verwendung

- **Arbeitsplatz-Ausstattung**: Erfasse Monitore als Teil der Arbeitsplatz-Ausstattung und verknüpfe sie mit dem zugehörigen Client.
- **Hardware-Inventar**: Dokumentiere Hersteller, Modell und Seriennummer für Warranty- und Asset-Tracking.
- **Standort**: Weise den Monitor einem Raum oder Arbeitsplatz zu.
