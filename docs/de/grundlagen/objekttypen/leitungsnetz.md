# Objekttyp: Leitungsnetz

Der Objekttyp **Leitungsnetz** dokumentiert logische Verkabelungssysteme, die mehrere physikalische Kabelverbindungen zusammenfassen. Er bietet eine übergeordnete Sicht auf zusammenhängende Verkabelungsabschnitte.

## Verwendung

- **Strukturierte Verkabelung**: Fasse zusammengehörende Kabel zu einem Leitungsnetz zusammen.
- **Gebäudeverkabelung**: Dokumentiere horizontale und vertikale Verkabelungssegmente.
