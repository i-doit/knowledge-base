# OCS Inventory NG

!!! info "Note"
    With the release of [i-doit 1.18.2](../versionshistorie/release-notes/release-notes-1.18/release-notes-1.18.2.md), we separated the previously integrated OCS functionality from the core. All content described here refers exclusively to i-doit versions < 1.18.2
