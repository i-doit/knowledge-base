This module requires the [**SHD VIEW Client**](https://smdocu.atlassian.net/wiki/spaces/SKB/pages/742260740/SHD+View+Desktop+Client) as a base module.

## Installation

The current version of the SHD Connect add-on can be downloaded from the following link: [Download](https://smdocu.atlassian.net/wiki/spaces/SKB/pages/6062081/Installation)

A list of the different versions including their changelog can be viewed at the following link: [SHD Connect](https://smdocu.atlassian.net/wiki/x/KwCcgw)

Once you have downloaded the add-on, you can easily upload and install it via the i-doit [Admin-Center](../../administration/admin-center.md) interface.
