# Connect Checkmk Add-on

The Connect Checkmk Add-on connects i-doit (CMDB and IT documentation) with Checkmk (network monitoring), so you can generate and synchronize your monitoring configuration directly from your CMDB data.

The add-on is developed and documented by [SHD System-Haus-Dresden GmbH](https://www.shd-online.de/). You can find the installation guide in the official SHD documentation:

[Installation guide (SHD)](https://smdocu.atlassian.net/wiki/spaces/SKB/pages/3467182081/Checkmk+-+Installation){ .md-button .md-button--primary }
