# Filling out forms

With the Forms add-on, forms can be created that users -- even without direct access to the i-doit interface -- can fill out. This allows standardized capture of requests for new hardware, onboarding processes, or change requests, for example.

## Procedure

1. Open the form via the provided link. The link is shared by the form creator.
2. Fill in the displayed fields. Required fields are marked accordingly.
3. Click **Submit** to send the form.

[![Filling out a form](../../assets/images/de/i-doit-add-ons/forms/ausfuellen/fillout.gif)](../../assets/images/de/i-doit-add-ons/forms/ausfuellen/fillout.gif)

After submission, a new object is automatically created in i-doit with the filled-in data. Which object type is created and which categories are populated depends on the form configuration.
