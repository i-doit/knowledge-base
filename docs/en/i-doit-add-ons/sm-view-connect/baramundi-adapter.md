# SHD Connect - Baramundi Adapter

With the Baramundi adapter for the SM-View Connect add-on, it is possible to import various objects from your Baramundi Management Suite into i-doit. Baramundi is a Unified Endpoint Management (UEM) solution that covers software distribution, patch management, and inventory management, among other things.

The adapter reads the devices and their properties recorded in Baramundi and creates them as objects in the i-doit CMDB or updates existing entries. This allows you to merge the automatically captured inventory data with your manual documentation in i-doit.

To use the adapter, you need a corresponding license, which you can obtain via the Marketplace of i-doit GmbH or directly from SHD.
