# Object Category VIVA

The [add-on](../index.md) **VIVA** provides a global category called **VIVA** that is available for all object types. This category gives the user a quick overview. It lists all VIVA-specific properties, including:

*   the information domain,
*   the target group with its primary IT-Grundschutz layer,
*   protection needs of the target group,
*   additional members of the target group,
*   modules assigned to the target group, and
*   statistics on the implementation of measures assigned to the target group.

If an object is assigned to multiple information domains, this is also recorded in the category. In addition, the user can navigate directly from the IT documentation in i-doit to the relevant areas of the VIVA add-on.

[![VIVA category in the CMDB module of i-doit](../../assets/images/de/i-doit-add-ons/viva/objekt-kategorien/1-vok.png)](../../assets/images/de/i-doit-add-ons/viva/objekt-kategorien/1-vok.png)

Assigning the category to object types is not necessary. It is automatically available for all object types. The category can be found above the category tree of each object, alongside other categories such as the CMDB Explorer or the logbook.
