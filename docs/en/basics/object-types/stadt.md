# Object Type: City

The **City** object type documents cities as a level in the location hierarchy. Cities are assigned to a country and contain buildings.

## Usage

- **Location Hierarchy**: Assign cities to a country and place buildings within cities.
- **Site Management**: Document different locations of the organization by city.
