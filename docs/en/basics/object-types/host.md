# Object Type: Host

The **Host** object type documents remote computers that do not directly belong to the organization's IT infrastructure but are relevant for documentation – e.g., external DNS servers or cloud services.

## Usage

- **External Systems**: Document external servers or cloud resources as host objects.
- **DNS Servers**: Record external DNS resolvers with IP address and operator.
- **Dependencies**: Link internal systems with external hosts that they use.
