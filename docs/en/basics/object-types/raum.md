# Object Type: Room

The **Room** object type documents physical rooms within buildings. Rooms contain racks, workstations, and other infrastructure components. They form an essential level in the location hierarchy.

## Usage

- **Location Hierarchy**: Assign rooms to a building and place racks and workstations within rooms.
- **Data Center Rooms**: Document server rooms and data center areas with their specific properties.
- **Access Control**: Record which persons have access to which rooms.
- **Infrastructure**: Assign air conditioning, emergency power, and UPS systems to a room.
