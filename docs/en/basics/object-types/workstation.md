# Object Type: Workstation

The **Workstation** object type documents logical workstations and links a person with their assigned devices (client, monitor, phone, printer). A workstation is a logical unit, not a physical object.

## Usage

- **Employee Equipment**: Document the complete equipment of a workstation – PC, monitor, phone, peripherals.
- **Person Assignment**: Link the workstation with the person using it.
- **Location**: Assign the workstation to a room for a complete overview.
- **Relocations**: Update location and device assignments during workstation moves easily in i-doit.
