# Object Type: Client

The **Client** object type documents end devices such as desktop PCs, laptops, and workstations. Clients are user workstations that are typically assigned to a person and a workstation. Unlike servers, the focus is on the user.

## Usage

- **End Device Inventory**: Record manufacturer, model, serial number, and hardware specifications of all clients in the organization.
- **User Assignment**: Link the client with the responsible person and the workstation – so it is always clear who uses which device.
- **Software and Licenses**: Document installed operating systems and applications. The link with licenses enables license compliance checks.
- **Support and Maintenance**: Record warranty expiration and maintenance contracts. Device data is immediately available for support requests.
- **Lifecycle Planning**: Use the Report Manager to identify all clients whose warranty is expiring or whose hardware is outdated.
