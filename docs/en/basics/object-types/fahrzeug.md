# Object Type: Vehicle

The **Vehicle** object type documents vehicles equipped with IT infrastructure or belonging to the IT organization. Typical use cases include company vehicles with telematics or mobile emergency vehicles.

## Usage

- **Vehicle Inventory**: Record vehicles with license plate, vehicle type, and responsible person.
- **IT Equipment**: Document IT devices installed in the vehicle, such as telematics units.
