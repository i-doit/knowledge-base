# Object Type: Network Zone

The **Net Zone** object type documents logical network zones for the segmentation of networks. Network zones group multiple subnets by security level or function.

## Usage

- **Network Segmentation**: Document network zones such as DMZ, internal LAN, or management network.
- **Subnet Assignment**: Assign Layer 3 networks to a network zone.
- **Security Policies**: Link network zones with firewall rules and access policies.
