# Object Type: Air Conditioning System

The **Air Conditioning** object type documents air conditioning systems in the data center. Air conditioning units are critical infrastructure components that maintain the operating temperature for servers and network components.

## Usage

- **Data Center Infrastructure**: Document air conditioning units as part of the data center infrastructure.
- **Location Assignment**: Assign the air conditioning unit to a room or building.
- **Maintenance Planning**: Record maintenance intervals and responsible service providers.
