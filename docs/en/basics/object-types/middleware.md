# Object Type: Middleware

The **Middleware** object type documents software that mediates between the operating system and applications – e.g., application servers, message brokers, or .NET Framework. Middleware is often a prerequisite for running applications.

## Usage

- **Middleware Inventory**: Record all deployed middleware components with version and installation location.
- **Dependencies**: Document which applications require the middleware.
- **Lifecycle Management**: Track end-of-support dates for used middleware versions.
