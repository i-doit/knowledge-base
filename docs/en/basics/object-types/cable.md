# Object Type: Cable

The **Cable** object type documents physical cable connections in the IT infrastructure. Each cable connects two ports and can be documented with details about type, length, and routing.

## Usage

- **Cabling Documentation**: Document every physical cable path with start and end point.
- **Cable Type and Length**: Record the cable type (Cat6a, OM4, etc.) and the length for maintenance and expansion planning.
- **Connection Diagrams**: Use cable objects as the basis for automatically generated topology diagrams.
