# Object Type: VRRP

The **VRRP** object type documents Virtual Router Redundancy Protocol configurations. VRRP provides automatic router failover by grouping multiple routers behind a virtual IP address.

## Usage

- **Router Redundancy**: Document VRRP groups with master and backup routers.
- **Virtual IP**: Record the virtual IP address shared by the VRRP group.
- **Network Resilience**: Use VRRP documentation for network high-availability analysis.
