# Object Type: System Service

The **System Service** object type documents background services and daemons running on servers. System services represent processes such as web servers (Apache, nginx), database services, or monitoring agents.

## Usage

- **Service Inventory**: Document all system services with their installation server and port.
- **Dependencies**: Record which applications or IT services depend on a system service.
- **Monitoring Integration**: Link system services with monitoring configurations.
