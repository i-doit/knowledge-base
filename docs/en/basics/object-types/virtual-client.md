# Object Type: Virtual Client

The **Virtual Client** object type documents virtual desktop machines. Virtual clients are VMs that serve as workstations and are typically provided via VDI solutions.

## Usage

- **VDI Inventory**: Document all virtual desktops with their hypervisor host.
- **User Assignment**: Link virtual clients with the assigned user.
- **Resource Allocation**: Record allocated CPU, RAM, and storage for each virtual client.
