# Object Type: VoIP Phone

The **VoIP Phone** object type documents IP-based telephones that communicate via the network. VoIP phones are assigned to a person, a workstation, and a phone system.

## Usage

- **VoIP Inventory**: Record all VoIP phones with model, MAC address, and extension number.
- **Network Integration**: Assign VoIP phones to VLANs and IP subnets.
- **Phone System Assignment**: Link VoIP phones with the SIP server or phone system.
