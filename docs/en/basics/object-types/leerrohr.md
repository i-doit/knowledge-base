# Object Type: Conduit

The **Conduit** object type documents conduits and protective pipes for cables. Conduits are installed in buildings and outdoors to allow cables to be pulled through later.

## Usage

- **Building Infrastructure**: Record conduits as part of the cable path infrastructure.
- **Cable Assignment**: Link pulled cables with the conduit.
- **Capacity**: Document available free conduits for future cabling.
