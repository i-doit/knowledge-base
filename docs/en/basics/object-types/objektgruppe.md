# Object Type: Object Group

The **Object Group** object type documents logical groupings of objects. Object groups have no technical function but serve to organize and structure the CMDB.

## Usage

- **Organizational Grouping**: Group objects by project, department, or any other criterion.
- **Overview**: Use object groups for dashboard widgets or reports to display a defined set of objects.
