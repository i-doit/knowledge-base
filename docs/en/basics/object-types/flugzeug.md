# Object Type: Aircraft

The **Aircraft** object type documents aircraft equipped with IT systems. This object type is specifically relevant for organizations in the aviation industry.

## Usage

- **Aircraft Inventory**: Record aircraft with identification and type.
- **IT Systems**: Document IT and avionics systems installed in the aircraft.
