# Object Type: Stacking

The **Stacking** object type documents switch stacks – multiple physical switches combined into a single logical unit. Stacking simplifies management and increases port capacity.

## Usage

- **Stack Management**: Document which switches are part of a stack and their roles (master/member).
- **Network Infrastructure**: Record stacking connections between the switches.
