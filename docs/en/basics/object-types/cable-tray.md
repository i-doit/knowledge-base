# Object Type: Cable Tray

The **Cable Tray** object type documents cable trays and cable ducts for the orderly routing of cables in buildings and data centers. Cable trays form the physical infrastructure for cable connections.

## Usage

- **Building Infrastructure**: Document cable trays as part of the building technology.
- **Route Planning**: Assign cables to a tray to document cable paths traceably.
- **Capacity**: Record the maximum occupancy of a tray.
