# Object Type: Printbox

The **Printbox** object type documents network print servers or print boxes that provide printing services in the network. Printboxes connect USB printers to the network.

## Usage

- **Network Infrastructure**: Document printboxes as part of the printing infrastructure.
- **Printer Assignment**: Link the printbox with the connected printers.
