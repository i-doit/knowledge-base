# Object Type: Amplifier

The **Amplifier** object type documents signal amplifiers for network or telecommunications signals. Amplifiers are used to boost signal strength over long cable runs.

## Usage

- **Signal Infrastructure**: Document amplifiers as part of the signal transmission chain.
- **Cable Connections**: Record connected cables on both sides of the amplifier.
- **Location**: Assign the amplifier to its installation location.
