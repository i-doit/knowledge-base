# Object Type: Converter

The **Converter** object type documents components for signal or media conversion, e.g., media converters from copper to fiber optic. Converters are used wherever different transmission media need to be connected.

## Usage

- **Network Infrastructure**: Record converters as part of the cabling infrastructure.
- **Cable Connections**: Document the connected cables on both sides.
- **Location**: Assign the converter to a rack or room.
