# Object Type: Replication Object

The **Replication Object** object type documents replication relationships between systems. Replication objects represent data synchronization between two or more systems.

## Usage

- **Replication Documentation**: Document replication partnerships between database servers, storage systems, or application servers.
- **Failover Planning**: Use replication objects for disaster recovery documentation.
