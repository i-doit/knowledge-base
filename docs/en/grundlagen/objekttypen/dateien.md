# Object Type: Files

The **Files** object type enables uploading and managing documents directly in i-doit. Files can be linked to any other objects and serve as a central document repository.

## Usage

- **Document Repository**: Upload manuals, datasheets, configuration backups, or certificates.
- **Versioning**: Maintain different versions of a document with version history.
- **Object Linking**: Assign files to any objects via the File Assignment category.
