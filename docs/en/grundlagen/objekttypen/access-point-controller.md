# Object Type: Access Point Controller

The **Access Point Controller** object type documents central controllers that manage and configure multiple wireless access points. The controller handles WLAN policies, roaming, and load balancing for all connected access points.

## Usage

- **WLAN Infrastructure**: Document the controller as the central management node of the WLAN infrastructure.
- **Access Point Assignment**: Link the controller with all managed access points.
- **Network and Location**: Assign the controller to its rack location and the management network.
