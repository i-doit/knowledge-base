# Object Type: Supernet

The **Supernet** object type documents supernets that group multiple Layer 3 networks (subnets) into a larger address range. Supernets provide an overview of the IP address space.

## Usage

- **IP Address Space**: Group subnets into supernets for a clear address hierarchy.
- **Network Planning**: Plan IP address ranges at the supernet level before creating subnets.
