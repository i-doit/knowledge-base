# Object Type: Printer

The **Printer** object type documents printing devices such as laser printers, inkjet printers, multifunction devices, plotters, and 3D printers. Printers can be assigned to a network, a room, or a workstation.

## Usage

- **Device Inventory**: Record manufacturer, model, serial number, and network addresses.
- **Location and Network**: Assign the printer to a room and an IP subnet.
- **Contact Assignment**: Document the responsible administrator or support contact.
- **Maintenance Contracts**: Link maintenance contracts directly with the device.
