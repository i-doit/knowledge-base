# Object Type: Uninterruptible Power Supply

The **UPS** object type documents Uninterruptible Power Supplies that protect IT equipment from power failures. UPS systems bridge short power outages and enable a controlled shutdown in case of longer failures.

## Usage

- **Power Infrastructure**: Document all UPS systems with capacity, runtime, and battery status.
- **Rack Assignment**: Assign UPS systems to racks and record the devices they protect.
- **Maintenance Planning**: Record battery replacement dates and maintenance contracts.
