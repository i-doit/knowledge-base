# Object Type: Person Groups

The **Person Groups** object type documents groups of persons such as departments, teams, or distribution lists. Person groups are used for contact assignments and permissions management in i-doit.

## Usage

- **Departments and Teams**: Document organizational units as person groups.
- **Contact Assignment**: Assign a person group as responsible for objects instead of individual persons.
- **Permissions**: Use person groups for role-based access control in i-doit.
