# Object Type: Virtual Host

The **Virtual Host** object type documents virtual machine hosts that run on a physical server. Virtual hosts represent the hypervisor layer or a virtualization cluster.

## Usage

- **Virtualization Management**: Document virtual hosts with hypervisor type and version.
- **Guest Systems**: Link the virtual host with all VMs running on it.
- **Resource Management**: Record available and used compute resources.
