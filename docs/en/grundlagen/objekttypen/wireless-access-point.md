# Object Type: Wireless Access Point

The **Wireless Access Point** object type documents Wi-Fi access points that provide wireless network access. Access points are typically managed by an Access Point Controller and assigned to a room.

## Usage

- **WLAN Infrastructure**: Document all access points with model, channel, and SSID.
- **Location Assignment**: Assign access points to rooms for coverage documentation.
- **Controller Assignment**: Link access points with the managing Access Point Controller.
- **Network Integration**: Document the wired uplink connection of the access point.
