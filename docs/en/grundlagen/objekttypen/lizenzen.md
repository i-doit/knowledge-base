# Object Type: Licenses

The **Licenses** object type documents software licenses. Each license object represents a license inventory for a software – with the number of licensed systems, expiration date, and license model.

## Usage

- **License Compliance**: Record all software licenses and compare licensed with used installations.
- **Expiration Tracking**: Monitor expiration dates of time-limited licenses and receive timely notifications.
- **Manufacturer Linking**: Assign licenses to a manufacturer object and a maintenance contract.
- **Reports**: Create reports on license compliance for audits and procurement planning.
