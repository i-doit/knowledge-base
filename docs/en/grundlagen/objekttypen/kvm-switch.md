# Object Type: KVM Switch

The **KVM Switch** object type documents components for remote provision of input devices (Keyboard, Video, Mouse). KVM switches enable the management of multiple servers via a single keyboard-monitor-mouse combination.

## Usage

- **Server Assignment**: Document which servers are connected to which ports of the KVM switch.
- **Rack Integration**: Assign the KVM switch to a rack.
- **Out-of-Band Management**: Document KVM switches as part of the out-of-band management infrastructure.
