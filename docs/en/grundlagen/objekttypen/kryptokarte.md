# Object Type: Crypto Card

The **Crypto Card** object type documents smart cards and Hardware Security Modules (HSM) with cryptographic functions. Crypto cards are used for digital signatures, encryption, and two-factor authentication.

## Usage

- **Security Infrastructure**: Document crypto cards as part of the security infrastructure.
- **User Assignment**: Assign crypto cards to the respective employee.
- **Certificate Tracking**: Record expiration dates of certificates stored on the card.
