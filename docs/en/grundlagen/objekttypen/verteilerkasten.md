# Object Type: Distribution Box

The **Distribution Box** object type documents electrical or network distribution boxes. Distribution boxes are typically installed on walls or in corridors and serve as junction points for cabling.

## Usage

- **Building Infrastructure**: Document distribution boxes as part of the building cabling.
- **Cable Connections**: Record the cables and connections within the distribution box.
- **Location**: Assign the distribution box to a room or building area.
