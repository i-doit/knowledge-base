# Object Type: Router

The **Router** object type documents network routers that forward data packets between different networks. Routers are central components of the network infrastructure.

## Usage

- **Network Infrastructure**: Document all routers with model, firmware version, and location.
- **Network Connections**: Record connections between routers and the networks they connect.
- **Routing Configuration**: Document routing tables and network interfaces.
- **Redundancy**: Record redundant router configurations for high availability.
