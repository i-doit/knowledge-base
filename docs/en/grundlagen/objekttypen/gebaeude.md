# Object Type: Building

The **Building** object type documents physical buildings that house IT infrastructure. Buildings are assigned to a city and contain rooms.

## Usage

- **Location Hierarchy**: Assign buildings to a city and add rooms within the building.
- **Address and Contact**: Record the address and responsible persons for each building.
- **Infrastructure Planning**: Record building-related infrastructure such as air conditioning units, distribution cabinets, and emergency power systems.
