# Object Type: Switch

The **Switch** object type documents network switches at Layer 2 and Layer 3. Switches are central network components that connect devices in a LAN.

## Usage

- **Network Infrastructure**: Document all switches with manufacturer, model, and firmware version.
- **Port Management**: Record all switch ports with VLAN assignment, speed, and connected devices.
- **VLAN Configuration**: Link switches with the VLANs they carry.
- **Stacking**: Document switch stacks and their member switches.
