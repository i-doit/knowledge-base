# Object Type: Telephone System

The **Phone System** object type documents telephone systems (PBX). Phone systems manage internal extensions and provide connectivity to the public telephone network.

## Usage

- **PBX Documentation**: Document phone systems with manufacturer, model, and installed modules.
- **Phone Assignment**: Link all phones that are connected to this phone system.
- **Network Integration**: Document SIP trunks and VoIP configurations.
