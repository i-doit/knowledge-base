# Object Type: Monitor

The **Monitor** object type documents display devices such as computer monitors, presentation displays, or video walls. Monitors are typically assigned to a workstation or a room.

## Usage

- **Display Inventory**: Record all monitors with manufacturer, model, screen size, and resolution.
- **Workstation Assignment**: Link the monitor with the corresponding workstation.
- **Port Connections**: Document connected cables and ports.
