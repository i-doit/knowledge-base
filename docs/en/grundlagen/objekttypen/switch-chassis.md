# Object Type: Switch Chassis

The **Switch Chassis** object type documents modular switch systems with a chassis that holds interchangeable line cards. Switch chassis are high-end switches used in data centers and campus networks.

## Usage

- **Modular Architecture**: Document the chassis with its slots and installed line cards.
- **Port Management**: Record ports across all line cards.
- **Network Infrastructure**: Assign the switch chassis to its location and connected networks.
