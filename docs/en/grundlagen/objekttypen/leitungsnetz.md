# Object Type: Wiring System

The **Wiring System** object type documents logical cabling systems that combine multiple physical cable connections. It provides a higher-level view of interconnected cabling sections.

## Usage

- **Structured Cabling**: Combine related cables into a wiring system.
- **Building Cabling**: Document horizontal and vertical cabling segments.
