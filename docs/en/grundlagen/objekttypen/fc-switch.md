# Object Type: FC Switch

The **FC Switch** object type documents Fibre Channel switches used in Storage Area Networks (SAN). FC switches connect servers and storage systems via the high-speed fiber optic protocol.

## Usage

- **SAN Topology**: Document FC switches as the central connectivity component between servers and storage systems.
- **Port Assignment**: Assign FC ports to the connected HBAs of servers or the storage system.
- **SAN Zoning**: Link the FC switch with SAN zoning objects for complete zone documentation.
- **Redundancy**: Document redundant FC switch paths for high-availability analysis.
