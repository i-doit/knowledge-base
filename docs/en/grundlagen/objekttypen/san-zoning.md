# Object Type: SAN Zoning

The **SAN Zoning** object type documents zoning configurations in Storage Area Networks (SAN). SAN zones define which servers can communicate with which storage ports.

## Usage

- **Storage Network**: Document SAN zones with their member ports.
- **Security**: Use zoning documentation for access control in the SAN.
- **Troubleshooting**: Maintain a complete view of SAN connectivity for troubleshooting.
