# Object Type: Power Distribution Unit

The **Power Distribution Unit (PDU)** object type documents rack-mounted power distribution units. PDUs distribute power to the devices in a rack and can provide monitoring data on power consumption.

## Usage

- **Power Distribution**: Document the power supply of rack devices via PDUs.
- **Rack Integration**: Assign PDUs to racks and record the connected consumers.
- **Energy Monitoring**: Track power consumption per outlet for capacity planning.
