# Object Type: Energy Supplier

The **Energy Supplier** object type documents external organizations that supply electricity. This object type allows you to fully map the dependency of your data center on a power provider.

## Usage

- **Supplier Documentation**: Record the name, contact details, and contract numbers of the power provider.
- **Dependencies**: Link buildings or data centers with the responsible energy supplier.
- **Emergency Planning**: Use the link for impact analyses in case of power outages.
