# Object Type: Telephone

The **Phone** object type documents landline telephones. Phones are assigned to a workstation and a phone system and can be documented with extension number and protocol information.

## Usage

- **Phone Inventory**: Record all phones with model, extension number, and assigned user.
- **Workstation Assignment**: Link phones with workstations for complete workplace documentation.
- **Phone System Integration**: Assign phones to a phone system.
