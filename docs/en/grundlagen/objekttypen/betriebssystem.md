# Object Type: Operating System

The **Operating System** object type documents operating system instances such as Windows Server, Ubuntu Linux, or VMware ESXi. An operating system object represents a specific OS version and is linked to the systems on which it is installed.

## Usage

- **OS Inventory**: Record all deployed operating systems with version, patch level, and support end date.
- **System Assignment**: Link the operating system to the servers, clients, or VMs on which it runs.
- **Patch Management**: Document the patch status and use the Report Manager to identify systems with outdated OS versions.
- **EOL Tracking**: Record the end-of-support date and receive early notifications about expiring operating systems.
