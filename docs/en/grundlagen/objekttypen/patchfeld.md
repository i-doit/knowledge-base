# Object Type: Patch Panel

The **Patch Panel** object type documents patch panels for the structured cabling of networks. Patch panels are installed in racks and serve as the central connection point between permanent building cabling and flexible patch cables.

## Usage

- **Structured Cabling**: Document patch panels as the interface between permanent and flexible cabling.
- **Port Assignment**: Assign individual ports to connected devices or outlets.
- **Rack Integration**: Assign the patch panel to a rack and record the occupied rack units.
