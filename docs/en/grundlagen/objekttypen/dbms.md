# Object Type: DBMS

The **DBMS** object type documents Database Management Systems such as MariaDB, PostgreSQL, Oracle, or MS SQL Server. A DBMS object represents the database engine installed on a server.

## Usage

- **Database Infrastructure**: Document all DBMS installations with manufacturer, version, and installation server.
- **Database Instances**: Link the DBMS with its associated database instances.
- **Dependencies**: Record which applications depend on a DBMS for impact analyses.
