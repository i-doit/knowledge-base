# Object Type: Contract

The **Contract** object type documents maintenance and support contracts. Contracts are assigned to manufacturers, suppliers, and devices and enable the tracking of contract terms and SLAs.

## Usage

- **Contract Management**: Record all maintenance contracts with duration, costs, and responsible supplier.
- **Device Assignment**: Link contracts with the covered devices or software licenses.
- **Expiration Dates**: Track contract expiration dates and identify renewal needs in time.
- **Reports**: Create reports on active and expiring contracts for budget management.
