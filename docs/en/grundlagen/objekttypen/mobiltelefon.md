# Object Type: Mobile Phone

The **Mobile Phone** object type documents cell phones, smartphones, and feature phones. Mobile phones are typically linked with a user and a SIM card.

## Usage

- **Mobile Device Inventory**: Record all smartphones with model, IMEI, and assigned user.
- **SIM Card Assignment**: Link the mobile phone with the inserted SIM card.
- **Mobile Contract**: Assign the associated mobile phone contract to the device.
