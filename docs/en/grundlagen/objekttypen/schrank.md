# Object Type: Rack

The **Rack** object type documents server racks and network cabinets. Racks hold other objects such as servers, switches, or patch panels and form the physical base structure of the data center in i-doit.

## Usage

- **Rack Layout**: Visualize the occupancy of a rack in rack units. i-doit shows a graphical view of installed components.
- **Location Assignment**: Assign the rack to a room to map the complete location hierarchy Country > City > Building > Room > Rack.
- **Capacity Planning**: Identify free rack units and plan the installation of new components.
- **Power Supply**: Link PDUs and UPS units that supply power to the rack.
- **Cable Management**: Document patch panels and cable trays within the rack.
