# Object Type: SIM Card

The **SIM Card** object type documents SIM cards for mobile communication. Each SIM card object can be linked to a mobile phone, a person, and a contract.

## Usage

- **SIM Card Inventory**: Record all SIM cards with phone number, provider, and contract details.
- **Device Assignment**: Link SIM cards with the mobile phone in which they are inserted.
- **Contract Assignment**: Assign SIM cards to the corresponding mobile phone contract.
