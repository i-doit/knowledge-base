# Managing Passwords

As a system administrator, you handle dozens of passwords every day. i-doit comes with a simple password management feature that keeps your head clear and helps you find needed credentials quickly.

## Distinction from Password Managers

i-doit is not a full-fledged password manager like [KeePass](https://en.wikipedia.org/wiki/KeePass). The motivation: to keep credentials as close to the [IT documentation](../glossar.md) as possible so you can find them quickly and easily in an emergency.

If you prefer to continue using a dedicated password manager, set links to the corresponding entries via the **Access** category. This keeps the credentials just a few clicks away.

Category **Passwords**
--------------------------

Any number of credentials can be stored per [object](../grundlagen/struktur-it-dokumentation.md). The [category](../grundlagen/struktur-it-dokumentation.md) **Passwords** is designed for this purpose. The data is stored encrypted in the respective [tenant](../administration/mandantenfaehigkeit.md) database of i-doit.
