# Release Notes 37
<!-- cSpell:disable -->

We are happy to announce the release of i-doit 37, the last version of the year 2025, focusing on **quality and compatibility**.

With this release, you can also assign icons to your custom categories in the category list. Additionally, we have officially supported **MySQL 8.4 LTS** compatibility.
A detailed list of all changes and improvements can be found in the [Changelog](../changelogs/index.md).

## Highlights in This Release

- More than 65 **quality improvements**
- New feature for adding **icons** to **custom categories**
- **Compatibility** with **MySQL 8.4 LTS**

## Add-ons

Alongside the release of i-doit 37, the following versions of our add-ons are also available:
- [API 2.5](../../i-doit-add-ons/api/index.md)
- [JDisc Connector 1.0.2](../../i-doit-add-ons/jdisc-connector/index.md)

Make sure to **update** the add-on accordingly to meet the requirements for i-doit 37.
