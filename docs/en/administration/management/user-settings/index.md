# User Settings

## The user settings are divided into the following sections

![index](../../../assets/images/de/administration/verwaltung/benutzereinstellungen/index-20260407-103057.png)

* [Password and Login](change-password.md) — The logged-in user can change their login credentials here.
* [Data Formats](datenformate.md) — The user can select various data formats.
* [User Language](user-language.md) — Under "User Language", the logged-in user can set a preferred user language.
* [User Interface](user-interface/index.md) — Here you can find the settings for the user interface
* [Edit Lock](edit-lock.md) — The edit lock prevents multiple users from editing the same object simultaneously.
