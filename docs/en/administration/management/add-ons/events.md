# Events

The Events [add-on](../../../i-doit-add-ons/events.md) enables a high degree of automation when working with the [IT documentation](../../../glossary.md). When something is changed in the IT documentation, the third-party system is informed about these changes. For example, when a new VM is documented in i-doit, it can be automatically created and provisioned on a virtualization host.
