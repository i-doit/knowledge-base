# Configuration

!!! info "This area has been replaced by the Replacement Add-on"
    The **Device Swap** function has been moved to the standalone **Replacement** add-on. The complete documentation on configuration and usage can be found there.

The device swap allows you to replace an existing object with a new one while automatically transferring category data, relationships, and other attributes. Via the configuration, you specify which object types are available for swapping, which CMDB status the swap device should receive, and whether category data should be kept or archived.

| Option | Value |
| - | - |
| Selectable object types | Categories |
| CMDB status for swap devices | CMDB status |
| Consider "Swap" status? | Yes or No |
| Archive swapped devices? | Yes or No |
| Also swap SYS-ID? | Yes or No |
| Keep category data of swapped objects? | Yes or No |
