# Device Swap

A new switch is being put into operation, but you want to keep all connections? Use the [==Device Swap== add-on](../../../../i-doit-add-ons/replacement.md) so you do not have to re-document all connections. This add-on enables the swapping of [objects](../../../../basics/it-documentation-structure.md) with configurable transfer of existing connections.
