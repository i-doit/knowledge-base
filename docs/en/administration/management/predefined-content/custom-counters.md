# Custom Counters

Here you can add custom counters to your i-doit.

You can reach this setting under **Administration → Predefined Content → Custom Counters**. With custom counters, you define how objects are automatically named and numbered when they are assigned to a specific object type. You can define prefixes, suffixes, the starting number, and the increment. For example, every new server automatically receives a sequential name such as "SRV-001", "SRV-002", etc. Use this function when you want to set up a uniform and automatic naming convention for specific object types.

A detailed guide can be found under [Custom Counters](../../../basics/custom-counters.md).
