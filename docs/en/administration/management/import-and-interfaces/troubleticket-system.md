# TroubleTicket System (TTS)

Here you can configure the TroubleTicket System (TTS) interface.

You can reach this setting under **Administration → Import and Interfaces → TroubleTicket System (TTS)**. Via this interface, you connect i-doit to an external ticket system such as Request Tracker (RT), OTRS, or iTop. You define the URL to the TTS and configure the linking parameters so that tickets can be referenced directly from within i-doit. When you enter a ticket ID in the "TroubleTicket" category of an object, a clickable link to the respective ticket in the external system is automatically generated. Use this function when you want to link your incident or service management with your IT documentation.
