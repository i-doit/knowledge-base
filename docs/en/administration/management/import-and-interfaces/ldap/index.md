# LDAP

!!! warning "Please create a complete backup before every change to an interface/import. If the result is not satisfactory, it can then be restored."

[![ldap-configuration](../../../../assets/images/de/administration/ldap-konfiguration.png)](../../../../assets/images/de/administration/ldap-konfiguration.png)

## LDAP is divided into the following sections

-   [Server](./server.md) — Set up your LDAP servers and test the connection.
-   [Directories](./directories.md) — Here the mapping attributes used for the LDAP interface can be specified.
-   [Attribute Extension](./attribute-extension.md) — With this function, you can customize the field names from the LDAP configuration file.
