# JDisc Configuration

!!! warning "Please create a complete backup before every change to an interface/import. If the result is not satisfactory, it can then be restored."

Under **Administration → Import and Interfaces → JDisc → JDisc Configuration**, you establish the connection between i-doit and your JDisc Discovery server. Enter the hostname or IP address of the JDisc server and the credentials for the JDisc database. Via the **Test connection** button, you can check whether the connection can be established successfully.

The JDisc integration allows you to import automatically discovered devices, network connections, and software installations from the JDisc database into the i-doit CMDB. Detailed documentation on the JDisc integration can be found under [JDisc Discovery](../../../../_archiv/jdisc-discovery.md).
