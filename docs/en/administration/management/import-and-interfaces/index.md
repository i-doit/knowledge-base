# Import and Interfaces

!!! warning "Please create a complete backup before every change to an interface/import. If the result is not satisfactory, it can then be restored."


[![verwaltung/import-overview.png](../../../assets/images/de/administration/verwaltung/import-uebersicht.png)](../../../assets/images/de/administration/verwaltung/import-uebersicht.png)

## Import and Interfaces is divided into the following sections

-   [Import Matching Profile](./import-matching-profile.md) — The object matching profiles were created to define criteria that are used when you want to establish a connection between an external object and an i-doit object.
-   [LDAP](./ldap/index.md) — Configure the LDAP connection for user authentication and synchronization.
-   [JDisc](./jdisc/index.md) — Import discovered devices from JDisc Discovery into i-doit.
-   [H-Inventory](./h-inventory.md) — Configure object matching profiles for the h-Inventory import.
-   [Monitoring](./monitoring/index.md) — Configure monitoring interfaces.
-   [TroubleTicket System (TTS)](./troubleticket-system.md) — Configure the TroubleTicket System interface.
-   [SMTP Configuration (Email)](./smtp-konfiguration-email.md) — Configure the mail server used by i-doit.
