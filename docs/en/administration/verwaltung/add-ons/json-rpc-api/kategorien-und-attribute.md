# Categories and Attributes

Here all available [attributes for the selected category](../../../../grundlagen/struktur-it-dokumentation.md) of i-doit are listed.

You can reach this overview under **Administration → Add-ons → JSON-RPC API → Categories and Attributes**. This page shows you a reference list of all categories with their respective attributes, as they can be accessed via the i-doit JSON-RPC API. For each category, you see the technical field names (constants) that you must use in API calls to read or write data. This overview is particularly helpful when you are developing your own integrations or scripts that communicate with i-doit via the API and you need the correct identifiers for categories and attributes.
