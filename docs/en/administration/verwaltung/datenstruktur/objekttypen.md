# Object Types

With this function, you can organize and edit the respective object types and their properties.

You can reach this setting under **Administration → Data Structure → Object Types**. Here you see a list of all existing object types, both the built-in ones and your own. You can edit existing object types, adjust their category assignment, change icons and colors, or create entirely new object types. The object type configuration determines which categories are assigned to an object type and how it is displayed in the object list and in the CMDB Explorer. Use this page when you want to adapt the standard object types to your documentation requirements or create custom object types for specific IT components.
