# Custom Categories

Here you can organize and edit custom categories.

You can reach this setting under **Administration → Data Structure → Custom Categories**. On this page, you see an overview of all self-created categories and can create new ones, edit existing ones, or delete them. Custom categories allow you to adapt the data structure of i-doit to your individual requirements by defining your own fields and field types. This way, you can document information that is not covered by the standard categories. Use this function when you need additional data fields for specific object types.
