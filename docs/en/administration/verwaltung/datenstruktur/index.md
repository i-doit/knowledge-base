# Data Structure

## The Data Structure is divided into the following sections


[![data structure](../../../assets/images/de/administration/datenstruktur.png)](../../../assets/images/de/administration/datenstruktur.png)

-   [Edit Data Structure](./datenstruktur-bearbeiten.md) — With this function, you modify the data structure of your tenant -- i.e., the assignment of object types to groups and of categories to object types.
-   [Object Types](./objekttypen.md) — With this function, you can organize and edit the respective object types and their properties.
-   [Custom Categories](./benutzerdefinede-kategorien.md) — Here you can organize and edit custom categories.
