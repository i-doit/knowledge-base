# User Language

Under **Administration → User Settings → User Language**, you can set the language of the i-doit interface for your user. This setting affects menus, category labels, system messages, and all other interface texts.

If you enable the option **Choose language via browser**, i-doit automatically adopts the language configured in the browser. If this option is disabled, the manually selected language is used instead. In addition to German and English, custom languages are also available, which can be created via [Language Profiles](../datenansicht/sprachprofile.md).

![benutzersprache](../../../assets/images/de/administration/verwaltung/benutzereinstellungen/benutzersprache-20260407-103241.png)

| Option | Value |
| ------ | ----- |
| Choose language via browser | Yes or No |
| Language | English<br>German<br>Custom language |
