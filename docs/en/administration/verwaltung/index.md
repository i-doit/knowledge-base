# Administration

Administration is the central area for all administrative tasks in i-doit. Here you configure the [IT documentation](../../glossar.md) via the web GUI -- from user settings to interfaces.

## Where do I find the Administration?

You can reach the Administration via the gear icon in your user area. This appears as soon as you hover over your login name in the upper right area.

[![Open Administration](../../assets/images/de/administration/verwaltung/index-20260407-103010.png)](../../assets/images/de/administration/verwaltung/index-20260407-103010.png)
