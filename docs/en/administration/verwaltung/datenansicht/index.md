# Data View

## The Data View is divided into the following sections

-   [Configure Object Browser](./objekt-browser-konfigurieren.md) — The object browser displays objects in a tabular format.
-   [Attribute Settings](./attribut-einstellungen.md) — With attribute settings, you define required fields, set advanced content defaults, and control the visibility of attributes.
-   [Language Profiles](./sprachprofile.md) — With custom translations, you can change the names of desired fields or add new languages with just a few clicks.
-   [Category Folders](kategorieordner.md) — With category folders, you organize categories within object types in a custom folder structure.
