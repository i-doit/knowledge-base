# Object Browser Configuration

The object browser displays objects in a tabular format. Via the configuration, you adapt the display to the needs of your users -- such as visible columns, sort order, and filter options.

[![Object Browser](../../../assets/images/de/administration/verwaltung/datenansicht/objekt-browser-konfigurieren/1-obk.png)](../../../assets/images/de/administration/verwaltung/datenansicht/objekt-browser-konfigurieren/1-obk.png)

First, select an attribute from a category that you want to configure for the object browser.

[![Object Browser](../../../assets/images/de/administration/verwaltung/datenansicht/objekt-browser-konfigurieren/2-obk.png)](../../../assets/images/de/administration/verwaltung/datenansicht/objekt-browser-konfigurieren/2-obk.png)

The selected attributes appear as columns in the object browser. You can select multiple attributes to get a comprehensive view of the object data.

[![Object Browser](../../../assets/images/de/administration/verwaltung/datenansicht/objekt-browser-konfigurieren/3-obk.png)](../../../assets/images/de/administration/verwaltung/datenansicht/objekt-browser-konfigurieren/3-obk.png)
