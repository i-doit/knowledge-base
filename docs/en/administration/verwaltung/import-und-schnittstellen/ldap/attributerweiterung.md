# LDAP Attribute Extension

With this function, you can customize the field names from the LDAP configuration file.

You can reach this setting under **Administration → Import and Interfaces → LDAP → Attribute Extension**. Here you can define additional LDAP attributes that go beyond the standard mapping. If your directory service contains custom attributes (e.g., custom schema extensions in Active Directory), you can add them here and assign them a readable field name in i-doit. This makes additional fields available during LDAP import that are automatically transferred to the person master data. Use this function when you want to map special attributes from your LDAP directory in i-doit.

Detailed documentation on LDAP integration can be found under [LDAP Directory](../../../../benutzerauthentifizierung-und-verwaltung/ldap-verzeichnis/index.md).
