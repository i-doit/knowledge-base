# Export Configuration

Define the [export parameters](../../../../i-doit-add-ons/nagios.md) for your monitoring system.

You can reach this setting under **Administration → Import and Interfaces → Monitoring → Export Configuration**. Here you define how i-doit exports the monitoring configuration (e.g., for Nagios or Checkmk). You can create and configure export profiles in which you specify the export path, type, and additional parameters. This allows i-doit to automatically generate the appropriate configuration files for your monitoring system from the monitoring data stored in the CMDB. Use this function when you want to maintain and automatically provision your monitoring configuration directly from the IT documentation.
