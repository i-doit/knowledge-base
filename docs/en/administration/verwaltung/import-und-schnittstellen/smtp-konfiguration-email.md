# SMTP Configuration (Email)

Under **Administration → Import and Interfaces → SMTP Configuration**, you set up the mail server through which i-doit should send emails. This configuration is required, among other things, for the notification system and for sending maintenance contract reminders.

Enter the hostname or IP address of your SMTP server and the associated port. If your mail server requires authentication, provide the username and password. Enable **SMTP TLS** if the connection should be encrypted. Via the **Sender** and **Name** fields, you specify from which address and under which display name emails are sent. With the **SMTP Debug** mode, you can enable detailed log output for troubleshooting.
