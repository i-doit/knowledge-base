# Request Tracker (RT)

[Request Tracker (RT)](https://bestpractical.com/request-tracker) is a web application for building and operating a service desk. RT and i-doit can exchange data via the RT extension [RT::Extension::ReferenceIDoitObjects](https://github.com/bheisig/rt-extension-referenceidoitobjects). This uses the [API](../../i-doit-add-ons/api/index.md) of i-doit to link tickets with one or more objects managed by i-doit. Within i-doit, all tickets linked to an object can be listed in the `**All Tickets**` category. Changes to ticket-to-object links are documented in the [logbook](../../grundlagen/logbuch.md) of i-doit. The extension supports [multi-tenancy](../../administration/mandantenfaehigkeit.md) of i-doit. A detailed description of setup and usage can be found on the [project website on GitHub](https://github.com/bheisig/rt-extension-referenceidoitobjects).
